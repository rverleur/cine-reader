# Sample Data

`TrimmedCine.cine` is a local large test file (packed 10-bit) used for smoke testing.

It is intentionally ignored by git via `.gitignore` (`sample_data/*.cine`) to avoid committing a 100MB+ binary file.

If you want to run examples/tests against this file, keep it at:

`sample_data/TrimmedCine.cine`
