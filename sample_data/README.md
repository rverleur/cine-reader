# Sample Data

`TrimmedCine.cine` is a local large test file used for examples and smoke
testing.

It is intentionally ignored by git via `.gitignore` (`sample_data/*.cine`) so
the repository and release source archives do not include a large binary test
file.

If you want to run the local examples and smoke tests against this file, keep
it at:

`sample_data/TrimmedCine.cine`

The repository and GitHub releases include only this README in `sample_data/`.
